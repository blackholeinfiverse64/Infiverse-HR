# Validation Tests

This directory contains comprehensive validation and testing files for the BHIV HR Platform.

## Files

### System Validation
- `complete_validation.py` - Complete system validation tests
- `comprehensive_test.py` - Comprehensive system testing
- `test_input_output_format_validation.py` - Input/output format validation

## Purpose

These tests ensure:
- System-wide functionality validation
- Input/output format compliance
- End-to-end system testing
- Data integrity validation

## Running Validation Tests

```bash
# Run complete system validation
python tests/validation/complete_validation.py

# Run comprehensive tests
python tests/validation/comprehensive_test.py

# Run I/O format validation
python tests/validation/test_input_output_format_validation.py
```
